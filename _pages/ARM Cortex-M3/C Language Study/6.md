# 숫자가 3 또는 5의 배수이면 1 아니면 0을 리턴하는 함수 설계
---

**문제**

```c
#include <stdio.h>

int f1(int num)
{
	// 코드 구현
}

void main(void)
{
	int num;
	scanf("%d", &num);
	printf("%d\n", f1(num));
}
```

```
입력 예시
15
```

```
출력 예시
1
```

**정답**
```c
#include <stdio.h>

int f1(int num)
{
	if (!(num % 3) || !(num % 5)) {
		return 1;
	}
	else {
		return 0;
	}
}

void main(void)
{
	int num;
	scanf("%d", &num);
	printf("%d\n", f1(num));
}
```
