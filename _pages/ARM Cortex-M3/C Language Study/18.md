# 입력 받은 정수 N(N > 0)의 자리수를 인쇄하는 코드를 설계하라
---

**문제**

```c
#include <stdio.h>

void main(void)
{
	// 코드 작성
}
```

```
입력 예시
12345
```

```
출력 예시
12345
```

**정답**
```c
#include <stdio.h>

void main(void)
{
	int num;
	int count = 0;
	scanf("%d", &num);
	while (num != 0) {
		num /= 10;  
		count++;
	}
	printf("%d", count);
}
```
