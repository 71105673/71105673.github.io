# 4자리 16진수 수를 입력 받아 각 자리 값을 인쇄하라 
---

**문제**

```c
#include<stdio.h>

void main(void)
{
	unsigned int x;
	unsigned int x4, x3, x2, x1;

	scanf("%x", &x);

	// 코드 작성

	printf("%X, %X, %X, %X", x4, x3, x2, x1);

}
```

```
입력 예시
AB9D
```

```
출력 예시
A, B, 9, D
```

**정답**
```c 
#include <stdio.h>

void main(void)답
{
	unsigned int x;
	unsigned int x4, x3, x2, x1;

	scanf("%x", &x);

	x4 = (x >> 12) & 0xF;  // 1000자리
	x3 = (x >> 8) & 0xF;   // 100자리
	x2 = (x >> 4) & 0xF;   // 10자리
	x1 = x & 0xF;		   // 1자리	

	printf("%X, %X, %X, %X", x4, x3, x2, x1);

}
```
